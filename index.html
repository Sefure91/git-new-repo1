Introduction About HTML 

HTML stands for hyper text markup language .it is used to create a web page.
HTML is the standard markup language for documents designed to be displayed in a web browser.
 It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Paragraph and Line break :


The difference between line break and paragraph is that line break breaks a line without altering the semantic structure of the page.
whereas paragraph sections the page into paragraphs. The element line break is an empty element in that.
although it may have attributes.
it can take no content and it may not have an end tag.



Body :


The body tag defines the document's body.
The body element contains all the contents of an HTML document.
such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
There can only be one body element in an HTML document.
